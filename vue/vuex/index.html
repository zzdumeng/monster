    module

    当使用 namespaced 为true 时, 需要注意的是 对于 state, 已经默认是添加了namespace, 而对于mutation, action, 和 getters, 都是以slash分隔开的.

    i.e.

    new Store({
  modules: {
    account: {
      namespaced: true,
      state: {
        name: ''
      },
      mutations: {
        change(state) {

        }
      }
    }
  }
})
{

  state: {}
}

    在组件内使用时, 对state就不能在mapState中使用字符串的形式, 而要写成函数的形式.

    {

computed: {
  ...mapState({name: state => state.account.name})
},
methods: {
  ...mapMutations({change: 'account/change'})
}
}
